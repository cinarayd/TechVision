# TechVision - Yazılım Şirketi Tanıtım Sitesi

TechVision, modern ve profesyonel bir yazılım şirketi tanıtım sitesidir. Siyah-beyaz renk teması ve yıldızlı arka plan tasarımıyla dikkat çeken, responsive bir web sitesidir.

## Özellikler

- Modern ve minimalist tasarım
- Responsive layout
- Yıldızlı animasyonlu arka plan
- 5 sayfalı yapı (Ana Sayfa, Hakkımızda, Hizmetler, Portfolyo, İletişim)
- Mobil uyumlu menü
- İletişim formu
- Google Maps entegrasyonu
- Sosyal medya bağlantıları

## Teknolojiler

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts

## Kurulum

1. Projeyi klonlayın:
```bash
git clone https://github.com/kullaniciadi/techvision.git
```

2. Proje dizinine gidin:
```bash
cd techvision
```

3. `index.html` dosyasını bir web tarayıcısında açın.

## GitHub Pages'de Yayınlama

1. GitHub'da yeni bir repository oluşturun.

2. Projeyi repository'ye push edin:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/kullaniciadi/techvision.git
git push -u origin main
```

3. Repository ayarlarından GitHub Pages'i etkinleştirin:
   - Settings > Pages
   - Source: main branch
   - Save

## Katkıda Bulunma

1. Bu repository'yi fork edin
2. Yeni bir branch oluşturun (`git checkout -b feature/yeniOzellik`)
3. Değişikliklerinizi commit edin (`git commit -am 'Yeni özellik: Açıklama'`)
4. Branch'inizi push edin (`git push origin feature/yeniOzellik`)
5. Pull Request oluşturun
